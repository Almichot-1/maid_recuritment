# Frontend Setup Instructions

## Required Package Installation

The accordion UI component requires an additional Radix UI package. Run this command in the `frontend` directory:

```bash
npm install @radix-ui/react-accordion
```

## Verification

After installing the package, verify all components compile without errors:

```bash
cd frontend
npm run build
```

## Development Server

To test the new features locally:

```bash
cd frontend
npm run dev
```

- `/candidates` - To test batch publish
